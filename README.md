

# 📊 Smart Expense Tracker

A Flask-based Personal Expense Tracker web application to manage income, expenses, and view financial reports in a simple dashboard format.

---

## 🚀 Features

* 📌 Dashboard showing total balance, income, and expenses
* ➕ Add new income/expense transactions
* 📊 Generate and view financial reports
* 💡 Simple and easy-to-use interface

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Frontend:** HTML, CSS
* **Database:** SQLite
* **Template Engine:** Jinja2

## ▶️ How to Run

```bash
git clone https://github.com/sinchanarokkad/Smart-Expense-racker.git
```

```bash
cd Smart-Expense-racker
```

```bash
pip install flask
```

```bash
python app.py
```

Then open:

```
http://127.0.0.1:5000/
```
